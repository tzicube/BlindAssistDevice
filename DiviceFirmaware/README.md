# DiviceFirmaware

Simple ESP32 project scaffold using PlatformIO and Arduino framework.

## Structure

- `src/`: application entry point
- `include/`: project headers
- `lib/`: reusable local libraries
- `test/`: unit tests

## Quick start

1. Install PlatformIO.
2. Open this folder in VS Code with the PlatformIO extension.
3. Build the project.
4. Upload to an ESP32 dev board.

## Default behavior

The sample firmware starts Serial at `115200` and blinks the built-in status LED on pin `2`.
